ABOUT TAGS

  • !DOCTYPE html: Declaration of the HTML5 doctype.
  • html: The root element of an HTML document.
  • head: Contains meta-information about the document.
  • meta: Provides metadata about the document, such as character encoding and viewport settings.
  • title: Sets the title of the webpage displayed in the browser's title bar.
  • style: Embedded CSS styles for styling the webpage.
  • nav: Represents navigation links.
  • header: Represents the header section of the webpage.
  • body: Contains the visible content of the webpage.
  • section: Represents a thematic section of content.
  • main: Contains the main content of the webpage.
  • footer: Represents the footer section of the webpage.
  • h1, h2, p, ul, li, a: Various text and link elements.
